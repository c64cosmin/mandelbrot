### Building

```
cargo build --release
```

### Running

```
./target/release/mandelbrot
```

###

You will see a menu on the top left
   
```
? - Open help message
Arrows- Move Slower
wasd  - Move
WASD  - Move Faster
+-    - Zoom
()    - Depth 7
1     - Color mode
2     - Outline mode
q     - Quit
```
